בנו RLC טורי עם מקור סינוס, נגד \(510\Omega\), קבל \(39nF\) וסליל. סרקו תדר עד למציאת מקסימום \(U_R\).

פתיחת Falstad Circuit Simulator

MATLAB Grader

חשבו \(f_0\), \(f_L\), \(f_H\), \(\Delta f\), \(Q\).

נגד\(f_0\)\(f_L\)\(f_H\)\(\Delta f\)\(Q\)
\(510\Omega\)
\(1000\Omega\)
% RLC series frequency response
R = 510; C = 39e-9; L = 0.1; % update L by measured value
f = logspace(2,5,800); w = 2*pi*f;
Z = R + 1i*(w*L - 1./(w*C));
H = R ./ Z; % UR/Vin
[~,idx] = max(abs(H)); f0 = f(idx);
plot(f,abs(H)); set(gca,'XScale','log'); grid on;
xlabel('f [Hz]'); ylabel('|U_R/V_{in}|');
דגש יחידות בניסוי 5: בחישובי תהודה הציבו \(L\) ב-H, \(C\) ב-F ו-\(f\) ב-Hz. אין להציב nF או mH ישירות ללא המרה. רוחב פס \(\Delta f\) נמדד ב-Hz, וגורם טיב \(Q=f_0/\Delta f\) חסר יחידות.
