חלק א': אימות משפט סופרפוזיציה ומאזן הספקים

בחלק זה נרכיב באופן מעשי את הרשת הדו-חוגית ונמדוד את השפעותיהם של מקורות המתח על זרם העומס בצורה נפרדת ומשולבת.

📋 מהלך המדידה והניסוי במעבדה

  1. הרכיבו על גבי לוח המטריצה את הנגדים הבאים: \(R_1 = 300\,\Omega\), \(R_2 = 1\text{ k}\Omega\), \(R_L = 1\text{ k}\Omega\).
  2. כוונו את ערוצי ספק הכוח DC למתחים הבאים: \(V_1 = 5\text{ V}\), \(V_2 = 3\text{ V}\). חברו אותם למעגל (ודאו אדמה משותפת!).
  3. מצב מלא (שני המקורות פועלים): הדליקו את שני הספקים. מדדו בעזרת רב-מודד דיגיטלי (במצב זרם DC) את הזרם הזורם דרך נגד העומס \(R_L\) (העבירו את רב המודד לטור לענף המרכזי). רשמו את הערך \(I_L\).
  4. מצב חלקי א' (רק V1 פעיל): כבו את המקור \(V_2\). חשוב: נתקו את החוט המחבר את \(V_2\) למעגל, וקצרו את קצה הנגד \(R_2\) ישירות לפס האדמה (GND). מדדו ורשמו את הזרם \(I'_L\).
  5. מצב חלקי ב' (רק V2 פעיל): כבו את המקור \(V_1\), החזירו את חיבורי \(V_2\). נתקו את \(V_1\) וקצרו את קצה הנגד \(R_1\) לאדמה. מדדו ורשמו את הזרם \(I''_L\).

📊 טבלת ריכוז המדידות (חלק א')

מצב עירור המעגל נוסחה תיאורטית זרם מחושב [mA] זרם נמדד בפועל [mA] סטייה יחסית [%]
מצב מלא: \(V_1\) ו-\(V_2\) פעילים \(I_L = I'_L + I''_L\)
מצב חלקי א': רק \(V_1\) פעיל (\(V_2\) מקוצר) \(I'_L\)
מצב חלקי ב': רק \(V_2\) פעיל (\(V_1\) מקוצר) \(I''_L\)
💾 טיפ הנדסי לחישוב סטייה: את הסטייה באחוזים חשבו לפי הנוסחה הבאה: \(\text{Error (\%)} = \frac{|I_{\text{measured}} - I_{\text{calculated}}|}{I_{\text{calculated}}} \cdot 100\). סטייה של פחות מ-5% נחשבת לתקינה במעבדה פיזית ומיוחסת לטולרנס (אי-דיוק מובנה) של הנגדים.
