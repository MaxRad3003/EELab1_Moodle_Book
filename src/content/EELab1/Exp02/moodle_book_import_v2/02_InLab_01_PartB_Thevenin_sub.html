חלק ב': משפט תבנין השקול והעברת הספק מקסימלי

בחלק זה נבצע מדידות בשני מצבי קיצון של המעגל—מצב חוג פתוח ומצב קצר מוחלט—על מנת לקבוע את הפרמטרים של מעגל תבנין השקול: מתח תבנין (\(V_{TH}\)) והתנגדות תבנין (\(R_{TH}\)).

חקירת המעגל ב-Falstad - תבנין והעברת הספק מקסימלי

הסימולציה מציגה את מעגל העבודה לחלק ב. השתמשו בה כדי לבדוק מתח חוג פתוח, זרם קצר והתנהגות העומס לפני המדידה במעבדה.

פתיחת סימולציית Falstad

מדידת מתח חוג פתוח, זרם קצר והסקת \(V_{TH}\) ו-\(R_{TH}\).

📝 שלבי העבודה ומהלך המדידה:

  1. נתקו לחלוטין את נגד העומס המרכזי (\(R_L\)). נקודות החיבור הריקות יוגדרו כהדקים A ו-B.
  2. מדידת מתח חוג פתוח (\(V_{OC}\)): חברו מד-מתח במקביל בין הדק A להדק B. רשמו את הערך (\(V_{TH} = V_{OC}\)).
  3. מדידת זרם קצר (\(I_{SC}\)): העבירו את המד למצב מדידת זרם, וחברו אותו ישירות בין A ל-B. רשמו את הערך.
  4. חישוב התנגדות תבנין: \(R_{TH} = V_{OC} / I_{SC}\).
