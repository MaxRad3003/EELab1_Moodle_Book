בחלק המעשי נאמת את משפטי הרשת ונמדוד התנהגות של רכיבים ליניאריים ולא-ליניאריים. עברו על החלקים לפי הסדר ותעדו את כל המדידות בטבלאות.

שיטת עבודה מומלצת

  1. בנו את המעגל כאשר הספק כבוי.
  2. קבלו אישור מדריך לפני הפעלת מתח.
  3. מדדו, רשמו יחידות, והשוו לתוצאה התיאורטית או לסימולציה.
  4. שמרו את הנתונים לשלב עיבוד MATLAB Grader ולדוח המסכם.
